# Retail-Sales-Analytics-Dashboard

**Tools Used**
Python
Pandas
NumPy
Matplotlib
Seaborn
Power BI

**Dataset**
Global Superstore Dataset
51,290 Transactions
24 Attributes

**Analysis Areas**
Sales Trend Analysis
Customer Segmentation
Product Performance
Regional Performance
Discount Impact Analysis
Logistics & Delivery Analysis

**Key Findings**
Technology generated the highest revenue and profit.
Consumer segment contributed the largest share of sales.
Central region was the top-performing market.
Higher discounts negatively impacted profitability.
Average delivery time was approximately 4 days.
